# Employee Exit Analysis Dashboard

This Power BI dashboard analyzes employee exit data and highlights key insights about employee attrition.

## Project Overview
The data was transformed and used to build an interactive dashboard in Power BI.

## Insights Included
- Total employees
- Total exits
- Exit count
- Exit reasons
- Employee exit trends
- Summary of attrition data

## Tools Used
- Power BI
- Excel / Data Transformation

## Project File
This repository contains the Power BI dashboard file for employee exit analysis.
